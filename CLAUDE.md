# The Cassini Mission Plan MCP

Read-only MCP server over the Cassini-Huygens `master_plan` dataset
(`data/cassini.db`, ~62k rows). Built as an AI workshop demo: clean,
readable, and runnable live in Claude Desktop.

**Stack:** TypeScript + Bun. **Transport:** local stdio.
